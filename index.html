compiler internals

A real language toolchain.
Not a wrapper.

Manta has a full compiler pipeline. Source code is lexed, parsed into an AST, type-checked, and emitted as TypeScript.

01 — Lexer
Tokenizer
Converts raw source text into a typed token stream with line/column tracking.
02 — Parser
Parser
Recursive-descent parser builds a typed Abstract Syntax Tree from the token stream.
03 — AST
Abstract Tree
Fully typed node system covering statements, expressions, blocks, and declarations.
04 — Codegen
TS Emitter
Walks the AST and emits idiomatic TypeScript with async/await and runtime imports.
05 — Runtime
Runtime Lib
TypeScript library implementing every built-in: detect(), track(), speak(), ask().
built-in capabilities

Everything wired up.
Out of the box.

No configuration files. No model setup. No audio API boilerplate.

📸
camera.start / .stop
One statement to open a camera stream. Works in browser via WebRTC or Node.js via OpenCV bindings.
👁️
detect <object>
Run YOLO or TensorFlow.js inference on the live feed. Returns confidence, bounding box, and distance.
🎯
track <object>
Multi-object tracking with persistent IDs. Powered by DeepSORT or ByteTrack under the hood.
🔮
predict trajectory
Kalman-filter trajectory prediction with a built-in collision_risk score from 0 to 1.
🗣️
speak "message"
Text-to-speech synthesis. Browser uses the Web Speech API; server uses a TTS engine.
🤖
ask "prompt"
Query a language model inline. Returns the response as a string you can use in your program.
⚡
when <condition>
Event-driven conditionals designed for real-time systems — cleaner than polling in a while loop.
🔌
sensor <device>.connect
Plug in ESP32, Arduino, or any serial sensor in one line. Read data with sensor.read().
🏗️
fn / var
Functions, variables, and control flow. Build reusable components without leaving the Manta mindset.
get started

Ship your first Manta
program in five minutes.

1
Clone the repo
git clone https://github.com/your-org/manta cd manta && npm install npm run build
2
Write a .mt file
# hello.mt
camera.start detect person if detected: speak "Person found"
3
Compile it
manta compile hello.mt ✓ Compiled → hello.ts
4
Or run directly
manta run hello.mt [manta] Camera started [manta] detect("person") → ✓ 94% [manta] speak: "Person found"
